# Automata

**Automata simulator written in C++ and SDL2**

Currently only one experimental implementation of a Game Of Life clone is implemented.

![Game Of Life](gol-preview.png)

## Prerequisites

### macOS

```
brew install sdl2
```


## Build

```
cmake -DCMAKE_BUILD_TYPE=Release .
cmake --build .
```
